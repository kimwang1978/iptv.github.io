# 🔧 Ray.so：开发者专属的代码截图与图标制作神器

在开发者分享项目或写博客的过程中，你是否也遇到过这些烦恼：

- 贴代码截图太丑、不统一？
- 想为项目做个图标却不懂设计？
- 社交媒体封面太普通，不够吸睛？

这时你就该试试 [**Ray.so**](https://ray.so) —— 一个由 Raycast 团队开发的在线工具，不仅可以生成精致的 **代码截图**，还提供了**图标生成器（Icon Maker）**，帮助你轻松创建出高颜值的开发者视觉素材。

---

## ✨ 什么是 Ray.so？

Ray.so 是一个专为开发者设计的**视觉内容生成工具**。最初以“代码截图工具”出圈，但现在已经拓展了更多功能，包括：

- 📸 高质量的代码截图
- 🎨 快速简洁的图标制作
- 🖼️ 项目封面图或社交分享图生成（测试中）

**无需注册账号，免费使用，打开即用！**

---

## 📸 代码截图功能

适合在以下场景使用：

- 教程中的代码展示
- 推文里的代码片段分享
- 项目介绍图文配图

特色包括：

- 多语言语法高亮（Prism.js 支持）
- 可切换背景色（黑、白、透明、紫等）
- 可选是否显示行号
- 一键导出高清 PNG

示例截图效果非常美观，远胜 IDE 截图或复制样式。

---

## 🧊 图标生成器（Icon Maker）

专为开发者和项目方设计的图标制作工具，适合：

- GitHub 项目图标
- Telegram / Notion / Discord 频道头像
- 技术栈徽标或博客页图标

特点如下：

- 输入 Emoji，自动生成图标（如 🐍 生成 Python 图标）
- 多种形状（圆角矩形、圆形、方形）
- 可选择配色、背景、阴影
- 可导出为 SVG 和 PNG

适合想省时又追求「好看」的开发者、博主、建站用户。

## 🔍 开源背景

Ray.so 的代码截图功能是开源的，GitHub 地址：

👉 [https://github.com/raycast/ray-so](https://github.com/raycast/ray-so)

主要技术栈：

- React + Next.js
- Tailwind CSS
- Prism.js 语法高亮
- 部署平台：Vercel

你可以基于此项目进行自定义开发或自托管。
